# Auto AI Lottery Research System

This project is a **data-driven simulation and analytics engine** for lottery-style systems such as:

- PowerBall (South Africa)
- PowerBall Plus (South Africa)

## ⚠️ Disclaimer
This system does NOT predict lottery outcomes.
Lottery draws are random and independent.

---

## 🚀 Features

- Data ingestion pipeline
- Feature engineering system
- Multiple AI models:
  - Frequency analysis
  - Markov chains
  - Bayesian smoothing
  - Monte Carlo simulation
- Automated training pipeline
- Simulation-based ranking system
- Streamlit dashboard
- Telegram notification bot (optional)
- GitHub Actions automation

---

## 🧠 Architecture

Pipeline:
1. Fetch data
2. Clean dataset
3. Train models
4. Run simulations
5. Generate ranked outputs
6. Push results to dashboard / Telegram
